HOW TO START THE WEBSITE
========================

OPTION 1: Double-Click (Easiest)
---------------------------------
  1. Open the "NALBAP-App" folder on your Desktop
  2. Double-click "Start.bat"
  3. Wait for the browser to open at http://localhost:3000
  4. Done! The website is running.

OPTION 2: From Visual Studio Code
-----------------------------------
  1. Open Visual Studio Code
  2. File > Open Folder > Select "NALBAP-App" on Desktop
  3. Press Ctrl+Shift+B (or Run > Start Without Debugging)
  4. Website opens at http://localhost:3000

OPTION 3: From Command Prompt
-------------------------------
  1. Open Command Prompt
  2. Type: cd Desktop\NALBAP-App
  3. Type: npm run dev
  4. Open browser to http://localhost:3000


HOW TO LET OTHERS TEST YOUR WEBSITE
=====================================

To let someone on the same WiFi/network test your website:

1. Find your PC's IP address:
   - Open Command Prompt
   - Type: ipconfig
   - Look for "IPv4 Address" (e.g., 192.168.1.100)

2. Give this URL to your client:
   http://YOUR-IP-ADDRESS:3000
   
   Example: http://192.168.1.100:3000

3. Make sure your PC's firewall allows port 3000
   (Windows may ask - click "Allow")

4. Your client opens that URL in their browser

TROUBLESHOOTING
================
  Problem: "Port 3000 already in use"
  Fix: Close other programs or restart PC

  Problem: "Module not found"
  Fix: Open Command Prompt in the NALBAP-App folder
       Type: npm install --legacy-peer-deps

  Problem: Blank page
  Fix: Clear browser cache (Ctrl+Shift+Delete)
